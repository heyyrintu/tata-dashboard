# Dashboard Template

A reusable, customizable dashboard template built with React, TypeScript, and Tailwind CSS. This template provides a modern design system with glassmorphism effects, theme support, and configurable components.

## Quick Start

1. **Copy the template** to your project directory
2. **Configure your dashboard** in `src/config/dashboard.config.ts`
3. **Customize theme** in `src/config/theme.config.ts` (optional)
4. **Set up API endpoints** in `src/config/api.config.ts`
5. **Build your dashboard** using the base components
